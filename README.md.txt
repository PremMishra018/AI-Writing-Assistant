# 🤖 AI Writing Assistant

An AI-powered writing assistant built using Python, Streamlit, and Google Gemini AI.

## ✨ Features

- Grammar Correction
- Professional Rewrite
- Friendly Rewrite
- Formal Email Generator
- WhatsApp Message Generator
- Text Summarizer
- Word Counter
- Character Counter
- Download Result

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
